# Tool 48
Патченное расширение Tool 42 для браузеров на базе Chromium с бесплатным премиумом.

> Tool 42 — проект, который включает в себя множество функций для ВКонтакте. Начиная от аналитики, заканчивая очисткой сообществ и персональной страницы.

### Установка
Установите распакованное расширение в браузер с помощью режима разработчика. (Поддерживаются все браузеры на Chromium, например, Chrome, Microsoft Edge, Opera, Vivaldi, Яндекс Браузер и т. д.)

||#### Chrome

Включите режим разработчика (1), нажмите "Загрузить распакованное расширение" (2) и выберите распакованную папку с файлами расширения.

## Важная информация

Я не являюсь разработчиком данного расширения. Все вопросы по работе функционала данного расширения задавайте в официальном сообществе: https://vk.ru/tool42_app. Исключением являются требование подписки VK Donut и пустая страница при нажатии на расширение, т. к. это может быть поломка расширения после патча кода (но такие случаи невероятно редкие) Перед тем, как задавать вопрос поддержке официального расширения протестируйте оригинальное расширение (https://chromewebstore.google.com/detail/tool-42/nncphlpcbkdenjngapbabbcommdljkmo) и сравните версию (внизу страницы). Если версия и проблема совпадают, то обращайтесь в поддержку официального расширения. Убедительная просьба: при обращении, пожалуйста, не упомянайте об этом репозитории.

Также, я гарантирую, что код этого расширения полностью (за исключением проверки подписки) совпадает с кодом оригинального, и отрицаю, что я краду ваши данные или занимаюсь другим видом мошеннической деятельности в отношении вас или вашего аккаунта ВКонтакте. Проверить код вы можете скачав `crx` файл официального расширения и сравнив его содержимое с этим репозиторием.

Для ознакомления: официальный разработчик говорит о некоторых проблемах с API ВКонтакте, из-за которых расширение может не работать должным образом (https://vk.ru/wall-173151748_4637, https://vk.ru/wall-173151748_4640, https://vk.ru/wall-173151748_4641)
